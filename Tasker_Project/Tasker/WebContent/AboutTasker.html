About Tasker

• User Management

Allows user
• To add new tasks
• To track his/her tasks
• To change status of the task and the changes should reflect in the lists dynamically
• To tag other users also in their task
• To search any task from a search bar in landing page
• To comment on the task along with other users


    How tasker is build

    • JSP/Servlet
    • HTML/CSS/Javascript
    • MySql
    • Eclipse
